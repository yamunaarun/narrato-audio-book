## How to Edit This Project

You can work on this project locally using your preferred IDE. Make sure Node.js and npm are installed.

### Steps to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/yamunaarun/scribe-whisper-studio.git

# 2. Navigate into the project folder
cd scribe-whisper-studio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
